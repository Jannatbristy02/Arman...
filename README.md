# SEO Specialist & Web Developer Portfolio Website

A professional, fully responsive portfolio website built with HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript.

## 🚀 Features

- **Modern Design**: Clean, professional, minimal design with soft green, beige, black, white, and warm orange accents
- **Fully Responsive**: Optimized for mobile, tablet, and desktop devices
- **Smooth Animations**: Fade-in animations on scroll and smooth scrolling navigation
- **Interactive Elements**: Hover effects, form validation, and dynamic content
- **SEO Optimized**: Semantic HTML5 structure with proper meta tags
- **Professional Sections**: All requested sections included with modern styling

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Open `index.html`** in your web browser
3. **Or use a local server**:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve . -p 8000
   
   # Using PHP
   php -S localhost:8000
   ```
4. **Visit** `http://localhost:8000` in your browser

## 🎯 Customization

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-green: #4CAF50;
    --accent-orange: #FF9800;
    /* Add your custom colors */
}
```

### Fonts
Change Google Fonts in the `<head>` section of `index.html`.

### Content
Update text content directly in the HTML file. Images use placeholder URLs - replace with your own assets.

### Contact Information
Update contact details in the footer and contact section.
